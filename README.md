UMLS RAT 
=========

UMLS **R**EST **A**PI **T**ool (RAT) provides a reasonable interface to the [UMLS Metathesaurus](https://uts.nlm.nih.gov/uts/umls/home) via the [REST API](https://documentation.uts.nlm.nih.gov/rest/home.html).
